# Release notes

## v0.9.2 – June 2026

Menu item links are now more flexible to configure. Minor bug fixes. Several internal libraries updated.

## v0.9.1 — May 2026

Replaced the color picker with a more reliable version. Some action buttons now remain visible when scrolling the page. Several internal libraries updated.

## v0.9.0 — May 2026

Partial redesign of the main layout. Improved news pages display. Search bars now look and behave consistently across the application. General responsiveness improvements.

## v0.8.5 — April 2026

Replaced an internal encryption library with a lighter, more actively maintained alternative. Improved error handling. Added autocomplete to login forms. Several internal libraries updated.

## v0.8.4 — April 2026

Improved navigation menu management: new item creation form, support for multiple item types, and various bug fixes. Newly created pages now automatically appear at the bottom of the list.

## v0.8.3 — April 2026

Fixed a bug with the media library file uploader. Several internal libraries updated.

## v0.8.2 — April 2026

Available languages can now be configured directly in the application settings.

## v0.8.1 — April 2026

Various fixes to the plugin system. Several internal libraries updated.

## v0.8.0 — March 2026

New plugin management page with improved activation and deactivation. Performance improvements.

## v0.7.7 — March 2026

Content management improvements: page search in the sidebar, ability to reorder pages, and support for link titles in page sections.

## v0.7.5 — March 2026

In development mode, raw data for content items is now accessible to help with debugging. Several internal libraries updated.

## v0.7.4 — February 2026

Responsive UI improvements for all screen sizes. Fixed a bug related to data structure settings. Several internal libraries updated.

## v0.7.3 — February 2026

Numerous improvements to the visual editor: better section field management and form generation.

## v0.7.2 — February 2026

Improvements to the initial setup wizard: enhanced database creation and bug fixes.

## v0.7.1 — February 2026

The icon picker in application settings can now be configured to return the icon name.

## v0.7.0 — February 2026

Complete overhaul of the access and authentication system with a cleaner architecture.

## v0.6.x — December 2025 – January 2026

Video metadata and visibility management. Bug fixes for the visual editor.

## v0.6.0 — December 2025

Launch of the visual editor: manage page sections directly from the interface via a dedicated sidebar.

## v0.5.0-rc.x — September–November 2025

Gradual improvements to the visual editor: data copying between sections, drag-and-drop reordering, preview mode, and better content model management.

## v0.5.0-rc.1 — September 2025

Major update: main layout redesign, media library improvements, and rich text editor upgrade.

## v0.0.17 — August 2025

Performance improvements: faster loading for articles, newsletters, and other content.

## v0.0.16 — May 2025

Introduction of the plugin system.

## v0.0.14 — April 2025

Auto-translation support in multilingual fields. User import and export functionality.

## v0.0.13 — March 2025

New field types in the form generator: date/time, time ranges, month, quarter, and week.

## v0.0.10 — March 2025

Added list and multilingual list fields to the form generator.

## v0.0.8 — February 2025

Mobile UI improvements. Configurable limit on multiple selections.

## v0.0.5 — January 2025

Numerous fixes and new features: permalink management, media picker, auto-updating sections, section position display, and pagination fixes.

## v0.0.2 — January 2025

First public release: dashboard with visits map and video management.
